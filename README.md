# Проєкт (частина С)

## Тема

ДЕЛЕГАТИ. ПОДІЇ

## Опис проєкту

Предметна область: **система управління проєктами компанії**.

У частині С проєкт було доповнено механізмами делегатів та подій для реалізації гнучкого пошуку, централізованої обробки повідомлень та системи сповіщень.

## Реалізований функціонал

*   **Делегати**:
    *   Власні: `WarningToMuchTasksInToDo`, `InfoMessage`, `ErrorMessage`.
    *   Standard .NET: `Action<T>`, `Predicate<T>`.
*   **Події**:
    *   `WarningToMuchTasksInToDoEvent`: попередження про > 5 завдань.
    *   `WarningToMuchToMuchTasksInToDoEvent`: попередження про > 10 завдань.
*   **MessagesHandler**: статичний клас для керування виводом повідомлень у консоль з різним кольором залежно від типу (Info, Warning, Error).
